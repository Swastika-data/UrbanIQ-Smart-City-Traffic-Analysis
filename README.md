##  Project Overview
UrbanIQ is a real-time smart city analytics platform designed to analyze traffic and weather data
to support city-level traffic optimization and decision-making.

##  Architecture
- Kafka for real-time data ingestion
- Spark Structured Streaming for processing
- Snowflake as cloud data warehouse
- Power BI for visualization

##  End-to-End Data Pipeline

1. Traffic & weather data ingestion via APIs
2. Real-time streaming using Kafka
3. Processing with Spark
4. Storage in Snowflake
5. Visualization using Power BI dashboards

## Key Insights
- Identification of congestion-prone routes using historical and streaming data
- Peak-hour delay analysis to highlight high-impact traffic windows
- Comparison of current speed vs free-flow speed to assess traffic efficiency
